# React Developer Intern Assignment

This repository contains a React application developed as part of the **React Developer Intern Assignment**.  
The project demonstrates core React concepts including component-based architecture, state management, routing, and responsive UI styling using Tailwind CSS.

## Steps to Run the Project Locally

### Prerequisites
- Node.js (version 18 or above)
- npm or yarn

Check Node.js version:
node -v

### Clone the Repository
git clone https://github.com/malleshmudigiri/Vgt-Assignment.git  
cd vgtassignment

### Install Dependencies
npm install

### Start the Development Server
npm run dev

Open the application in your browser:
http://localhost:5173

## Tech Stack
- React (Vite)
- React Router DOM
- Tailwind CSS
- JavaScript (ES6+)
- LocalStorage API
